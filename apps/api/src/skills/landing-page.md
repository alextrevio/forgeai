# Landing Page Skill

## Objetivo
Crear una landing page moderna, atractiva y de alta conversión con animaciones suaves y diseño responsive.

## Stack
- React + Vite (o HTML/CSS estático si se pide simple)
- Tailwind CSS
- Framer Motion para animaciones (o CSS animations)
- Lucide React para iconos

## Secciones Estándar

1. **Hero Section**
   - Headline principal (H1) claro y conciso
   - Subtítulo explicativo
   - CTA principal (botón grande y visible)
   - Imagen/ilustración hero o mockup del producto
   - Gradient o pattern de fondo sutil

2. **Social Proof / Logos**
   - "Usado por empresas como..." con logos
   - O métricas: "10K+ usuarios", "99.9% uptime"

3. **Features / Beneficios**
   - Grid de 3-4 features con iconos
   - Título + descripción corta por feature
   - Alternativa: Feature sections con imagen lateral

4. **How It Works**
   - 3 pasos numerados con iconos
   - Flujo visual claro

5. **Testimonials**
   - 2-3 testimonios con foto, nombre, cargo
   - Rating con estrellas si aplica

6. **Pricing** (si se requiere)
   - 2-3 planes en cards
   - Plan recomendado destacado
   - Feature comparison list

7. **FAQ**
   - Accordion expandible
   - 4-6 preguntas comunes

8. **CTA Final**
   - Repite el call-to-action principal
   - Formulario de email o botón de registro

9. **Footer**
   - Logo, links de navegación
   - Redes sociales, legal, copyright

## Design Guidelines
- Dark mode por default (#0A0A0A fondo)
- Color accent llamativo para CTAs
- Tipografía: headlines grandes (text-4xl a text-6xl), body legible (text-base a text-lg)
- Spacing generoso entre secciones (py-20 a py-32)
- Animaciones de entrada: fade-in-up al hacer scroll
- Mobile-first: todo debe verse perfecto en 375px
- Imágenes con aspect-ratio fijo y lazy loading
- Máximo contraste en CTAs vs background

## SEO
- Semantic HTML (header, main, section, footer)
- Meta title y description optimizados
- Alt text en todas las imágenes
- Estructura de headings jerárquica (H1 > H2 > H3)
- Carga rápida: optimizar assets
